# swingy
 text-based RPG game in Java (Console and GUI)

# usage
Make sure to have [Maven](https://maven.apache.org/) installed.
For console:
```bash
mvn clean
mvn package
java -jar swingy.jar console
```
For GUI:
```bash
mvn clean
mvn package
java -jar swingy.jar gui
```

You can then change the view inside the program.
